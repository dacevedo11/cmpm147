Description

The goal of this experiment was to create a grid that places tiles using autotile. Once the first one worked, we were supposed to create a second grid that was a dungeon if you created an overworld, or an overworld if you created a dungeon. Then, we were supposed to add variety to make it more interesting and add some life to it, like animated tiles with moving water or such.
