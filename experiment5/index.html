Description

The goal for this assignment is to build a generator that directly reads inspiring images, renders parameterized images, and adjusts its own designs to optimize an evaluation score.

Technical

This project takes in an image selected from the drop down menu. It reads the images and uses draw functions to draw the design. I did this with rectangles. The design is mutated based on a mutation rate that can be adjusted using the slider.

Reflection

This project wasn't as difficult or as time consuming as the others so far. It was nice having Wes's example available too. I did some playing around with the paramaters, and I think I got some interesting results. I also messed around with the number of shapes and the sizes of the rectangles. I believe my experiment should meet the requirements of the assignment. I'm mostly happy with the results.
